### About Notebook & Environment

To run this notebook, either 
1.) follow the instructions to use the Spell platform or 
2.) follow these steps to run locally:  
    a. Set up a local environment to install all model notebook dependencies. You can create local environment using the following commands (Python 3 & Windows): 
    To create virtual environment with modelenv folder in the root directory:
    - python -m venv model_environment_name; eg: python -m venv modelenv 
    To activate the model virtual environment:
    - modelenv\Scripts\activate    
    b. Install all the dependencies mentioned in requirements.txt in a local environment using following command:
    - python -m pip install -r requirements.txt
